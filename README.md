# 🧑‍💻 Coding Test Study
코딩 테스트 실력 향상을 목표로 한 스터디의 문제 풀이 기록 저장소입니다!  
SSAFY 부울경 1반, 2반의 인원이 함께 합니다.

## 👩‍👩‍👧‍👦 Members
참여 인원: 김경린, 박은서, 차서영, 황순철

---

## 📂 Directory Structure

```bash
{이름}/
 ├─ solved/     # 해결한 문제
 └─ unsolved/   # 시도 중 / 실패한 문제
```

## 🏷️ File Naming

```bash
BOJ_문제번호.확장자
```

## ⭐ Commit Convention   

```bash
[type] [BOJ-문제번호] 문제이름 - 이름
```

| type | 의미 | 설명 |
| --- | --- | --- |
| solved | solve, unsolved → solved |   • 최초 1회로 문제 해결완료 
  • 최초 1회로 해결하지 못한 문제를 해결했을 때 |
| unsolved | unsolved | 최초 1회 문제 풀이 실패 |
| refactor | 로직 개선 (정답 유지) | 이미 푼 문제지만, 클린 코드를 만들 때 사용 |
| feedback | 타인 피드백 반영 |  |


- 예시
    
    ```bash
    # 10870 피보나치 수 문제를 해결했다.
    [solved] [BOJ-10870] 피보나치 수 - 김경린
    
    # 10773 제로 문제를 풀었으나 해결하지 못했다.
    [unsolved] [BOJ-10773] 제로 - 김경린
    
    # 10773 제로 문제를 뒤늦게라도 해결했다.
    [solved] [BOJ-10773] 제로 - 김경린
    ```
